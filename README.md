# Single User Cinema System

A Java console application that allows a single user to manage cinema seat reservations.

## Features

- Password authentication
- View seat status
- Reserve seats
- Cancel reservations
- Change PIN
- View last 3 actions
- Input validation

## Technologies

- Java
- Scanner
- Arrays and Strings
- Console-based UI

## How to Run

Compile:

javac SingleUserCinemaSystem.java

Run:

java SingleUserCinemaSystem
